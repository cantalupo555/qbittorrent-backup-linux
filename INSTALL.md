# Manual Installation (Optional)

```bash
# Clone the repository
git clone https://github.com/cantalupo555/qbittorrent-backup-linux.git
cd qbittorrent-backup-linux

# Install to system
sudo ./install.sh
```

## Uninstall

```bash
sudo qbittorrent-backup --uninstall
```

## Usage

After installation, run with:

```bash
# Interactive menu
sudo qbittorrent-backup

# Direct backup
sudo qbittorrent-backup backup

# Direct restore
sudo qbittorrent-backup restore

# Show help
sudo qbittorrent-backup -h

# Show version
sudo qbittorrent-backup -v
```
